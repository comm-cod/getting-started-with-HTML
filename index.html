Purpose of comments in HTML

The purpose of comments in HTML is to explain your code, which will help you or others to edit the source code at a later date.


Example of how to use comments :-

To leave a comment in HTML, place a < !-- tag before the code and a -- > tag after the codee that you want to hide.

a short note on Tag and Element with an Example

Tag :- a tag is a piece of markup language used to indicate the beginning and end of an HTML element in an HTML document.
For example: <h1> and </h1> are two tags indicating beginning and ending of a h1 tag.
Element :- An HTML element is defined by a starting tag. If the element contains other content, it ends with a closing tag.
For example: < p> is starting tag of a paragraph and < /p> is closing tag of the same paragraph but < p>This is paragraph< /p> is a paragraph element.

What is the DOCTYPE Declaration in HTML?

The HTML document type declaration, also known as DOCTYPE , is the first line of code required in every HTML or XHTML document. The DOCTYPE declaration is an instruction to the web browser about what version of HTML the page is written in. This ensures that the web page is parsed the same way by different web browsers.
